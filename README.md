us-weather
================================================================================

API providing weather data from <http://graphical.weather.gov/xml/rest.php>

usage
--------------------------------------------------------------------------------

Add "us-weather" to your package.json `dependencies` property, then
`npm install` in your project to install it.  Or whatever.

--------------------------------------------------------------------------------
